# AI Content Generator

🚀 A modern AI-powered SaaS application for generating high-quality content, marketing copy, and SEO-optimized text.

## Features

* Generate blog posts
* Create social media content
* Generate product descriptions
* Write email marketing campaigns
* Create SEO meta descriptions
* Multiple content tones
* Responsive dashboard
* Modern SaaS UI/UX

## Content Types

* Blog Articles
* Social Media Posts
* Product Descriptions
* Email Campaigns
* SEO Content
* Ad Copy
* Video Scripts

## Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS
